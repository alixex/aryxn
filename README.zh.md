## **产品简介**

**Aryxn** 是一个以"用户主权"为核心的 Web3 应用，将**多链账户管理**、**永久文件存储**与**历史检索能力**融合为一个统一平台。  
名字源自柏拉图的"回忆说（Aryxn）"，帮助你掌控数字资产，并永久守护那些不该丢失的重要数据——安全、私密、永恒。

**🌐 在线地址**：https://alixex.github.io/aryxn/

---

## **核心功能**

### 🔐 **多链账户管理**

- **统一账户中心**：在一个安全的地方管理 **Ethereum**、**Solana**、**Sui**、**Arweave**、**Bitcoin** 等多个主流区块链的账户
- **双重钱包支持**：使用内部账户（通过主密码本地加密）或连接 MetaMask、Phantom 等外部钱包
- **实时余额**：查看所有链上的代币余额和账户详情，自动同步更新
- **安全设计**：私钥和敏感数据通过主密码本地加密，永远不会离开你的浏览器
- **轻松导入**：通过私钥或助记词导入现有账户，或一键创建新账户

### 📁 **永久文件存储**

- **永久存储**：将文件上传到 Arweave 区块链，实现永久、不可篡改的存储，只要网络存在，文件就永久保存
- **灵活隐私**：选择**公开存储**供开放访问，或**私密加密存储**保护敏感文件
- **端到端加密**：文件在上传前使用主密码在本地加密，只有你拥有解密密钥
- **智能组织**：使用文件夹、标签和自定义描述组织文件，轻松管理
- **强大搜索**：通过全文搜索和高级筛选功能，按名称、类型、日期或标签快速找到文件
- **通用访问**：使用去中心化文件链接从任何设备访问文件，数据随你而行
- **上传成本优化**：实时费用估算与路径引导，尽量降低上传成本

### 📊 **统一仪表板**

- **全面概览**：在一个界面中查看上传活动、文件记录与状态
- **文件历史**：追踪文件相关操作，包含时间、元数据、大小与存储网络信息
- **存储分析**：监控总上传文件数、存储使用量和上传历史
- **跨链同步**：自动将链上数据同步到本地 SQLite 缓存，实现快速访问和离线查看
- **高级筛选**：在所有交易类型中搜索和筛选，提供强大的查询能力

### 🔎 **搜索与检索**

- **统一搜索**：支持按文件名、元数据和交易 ID 检索
- **桌面/移动端搜索体验**：分别提供适配的搜索交互
- **资源直达访问**：通过资源链接与网关回退进行打开/下载

## **隐私与安全**

### 🔒 **你的数据，你做主**

- **本地优先架构**：所有敏感信息（私钥、主密码、加密数据）都存储在浏览器本地并加密，永远不会发送到任何服务器
- **零知识设计**：Aryxn 永远不会收集、存储或访问你的数据。即使我们想，也看不到你存储的内容
- **默认隐私**：设计上尽量减少中心化依赖并降低信任假设，让你始终保持完全控制
- **浏览器原生**：整个应用完全在浏览器中运行，没有后端服务器处理你的敏感数据

### 🛡️ **加密与保护**

- **端到端加密**：文件使用行业标准加密在本地加密后上传，只有你能解密
- **主密码保护**：单一主密码保护所有账户和加密文件，一把钥匙管理所有数据
- **二次验证**：查看私钥或导出账户等敏感操作需要额外密码确认
- **默认安全**：所有敏感操作都需要明确的用户确认，避免意外泄露数据

## **适用场景**

### 🔐 **多链账户管理**

在一个界面中管理不同链上的账户，切换活跃身份，并通过本地加密保护密钥与敏感信息。

### 📦 **个人数字保险库**

构建你的个人数字档案，在区块链上安全存储票据、证书、照片和重要文档。对敏感文件使用加密功能，确保你的私人信息永久保持私密。

### 🎨 **创作者存证与知识产权保护**

在区块链上发布并时间戳标记你的创作作品，获得永久可验证的创作证明。适合需要不可篡改的所有权和创作日期证明的艺术家、作家、音乐家和创作者。

### 🧭 **数据优先的 Web3 工作流**

在同一应用里完成账户管理、公开/加密上传和历史检索，形成高效闭环。

### 🗄️ **长期数据备份**

创建关键文件的永久备份，这些备份将超越任何单一公司或服务。一旦上传到 Arweave，你的文件将永久存储，受到去中心化基础设施和密码学保障的保护。

## **快速开始**

1. **访问** https://alixex.github.io/aryxn/
2. **创建** 主密码以加密你的数据（请务必记住它，这是访问加密数据的唯一方式）
3. **设置账户**：
   - 通过私钥或助记词导入现有账户
   - 为支持的区块链创建新账户
   - 或连接 MetaMask、Phantom 等外部钱包
4. **开始使用核心流程**：
   - **管理账户**：在支持链上维护账户与身份
   - **上传文件**：选择公开或加密上传模式
   - **搜索检索**：在仪表板与搜索入口快速定位资源
5. **上传文件**：
   - 选择公开或加密存储
   - 文件自动路由到你的 Arweave 账户
   - 如有需要，系统会提供支付路径建议
6. **全面追踪**：从统一仪表板监控上传记录、历史状态与存储分析

## **技术栈**

- **前端**：React + Vite + TypeScript + Tailwind CSS
- **状态管理**：Zustand 配合本地持久化
- **多链 SDK**：自研 wallet-core 集成 Ethereum、Solana、Sui、Bitcoin、Arweave
- **存储**：Arweave 永久文件存储 + 本地 SQLite 快速索引
- **加密**：本地使用 AES-256-GCM 加密，端到端加密文件上传

## **Monorepo 架构**

### **apps/**

- **vault/**：面向用户的主 Web 应用（账户管理、上传、仪表板、搜索、设置）

另见：[apps/README.zh.md](apps/README.zh.md)

### **packages/**

- **arweave/**：Arweave 交互能力与集成逻辑
- **chain-constants/**：共享链配置、代币元数据与路由常量
- **changelogs/**：包级变更说明资产（用于发布/文档流程）
- **cross-chain/**：跨链桥接编排与状态处理
- **crypto/**：共享加密原语与编码工具
- **exchange-chain/**：多链兑换路径规划引擎
- **query-chain/**：多链数据查询抽象层
- **storage/**：持久化/本地存储辅助能力
- **swap-ethereum/**：Ethereum 侧兑换执行适配
- **swap-multichain/**：多链兑换协调层
- **swap-solana/**：Solana 侧兑换执行适配
- **wallet-core/**：钱包核心生命周期、账户操作与签名抽象

另见：[packages/README.zh.md](packages/README.zh.md)
